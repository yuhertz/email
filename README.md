# Python Email

This sends email faster using python.

Set up for Kali Linux
```
sudo apt install git
sudo apt install python3
git clone https://github.com/yuhertz/email
cd email
python3 email.py
```

Set up for Termux
```
pkg install git
pkg install python3
git clone https://github.com/yuhertz/email
cd email
python3 email.py
```
